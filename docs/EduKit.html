edukit.freqeuncyToArray(counter)[소스]¶

명사 빈도수 데이터를 차트의 각각 축제목, y값, x값 으로 변환해줍니다.

매개변수

counter (Counter형) – 명사들이 담긴 리스트

반환값

축제목, y값, x값

반환 형식

Counter형

>>> text = '우리학교는 산에있는 학교이다.'
>>> nouns = getNouns(text)
>>> top_nouns = frequnecyNouns(nouns)
>>> freqeuncyToArray(top_nouns)
['우리학교, '산', '학교'], [1, 1, 1], [1, 2, 3]
edukit.frequnecyNouns(nouns, max_result)[소스]¶

명사들의 빈도수를 분석합니다.

매개변수

  • nouns (리스트형) – 명사들이 담긴 리스트

  • max_result (정수형) – 상위 n개를 추출함

반환값

문장안에 있는 모든 명사들의 갯수

반환 형식

Counter형

>>> text = '우리학교는 산에있는 학교이다.'
>>> nouns = getNouns(text)
>>> frequnecyNouns(nouns)
Counter[('우리학교', 1), ('산', 1), ('학교', 1)]
